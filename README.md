# 🚗 COS40005 Computing Project A – Smart Parking System

## 📌 Overview
This repository contains a **Smart Parking System** developed as part of **COS40005 – Computing Project A**.  
The project integrates **computer vision**, **Internet of Things (IoT)**, and **backend software systems** to deliver an **intelligent, automated parking management solution**.

The system leverages **advanced machine learning techniques** for vehicle identification and combines them with **real-time sensor-based monitoring** to manage parking space availability, vehicle entry, and exit operations efficiently.

In addition to the embedded and AI components, the system is designed to support:
- A **backend database** for storing vehicle, membership, and parking session data
- A **web-based interface** for system monitoring and administration
- A **mobile application** for registered members to manage access and view parking information

The overall architecture follows a **modular and scalable design**, with a clear separation between:
- **AI-based license plate recognition**
- **IoT-based hardware control**
- **Backend services and user-facing applications**

This separation allows each subsystem to be developed and extended independently while functioning as a unified smart parking solution.
